# Проект Grocery Store

## Описание проекта
Grocery Store - это проект магазина продуктов.

Пользователю необходимо зарегистрироваться и затем авторизироваться, чтобы у была возможность добавлять продукты в корзину покупок и получать результирующий список. Неавторизированным пользователям будет доступен только просмотр категорий с подкатегориями и продуктов.

Категории, подкатегории и продукты может добавлять только администратор.

**В проекте доступно**:  
*для всех пользователей*  
- просмотр категорий  
- просмотр подкатегорий  
- регистрация  
- авторизация  

*для авторизированных пользователей*  
- просмотр категорий с подкатегориями  
- добавлять/удалять рецепты в/из корзину(-ы)  

*для администратора доступно всё, что указано выше и ещё*  
- редактирование, редактирование и удаление любых продуктов (в админке)  
- добавление, удаление и редактирование категорий (в админке)  
- добавление, удаление и редактирование подкатегорий (в админке)  


## Пользовательские роли
**Аноним** — доступен просмотр категорий, подкатегорий и продуктов.  
**Аутентифицированный пользователь (user)** — доступно добавление продуктов в свою корзину покупок, а также удаление и редактирование продуктов в ней.  
**Администратор (admin)** — доступны права администратора, также доступно создание продуктов, категорий и подкатегорий (в админке).  



## Используемые технологии
*Проект реализован с использованием следующего функционала*: 
- Python 3.12
- Django 5.1.1 
- Django REST Framework 3.15.2

Для аутентификации в Grocery Store используются JWT (SimpleJWT) + Djoser для регистрации.  
Все необходимые для работы Foodgram зависимости перечислены в requirements.txt.  
Рекомендуется перед установкой развернуть виртуальное окружение с верисей python 3.12:  

Bash:
```
py -3.12 -m venv venv 
```

Для установки зависимостей необходимо выполнить установку зависимостей:
```
pip install -r requirements.txt
```


## Описание API
API предоставляет следующие возможности:

**Пользователи**
- Регистрация пользователей
- Аутентификация пользователей

**Категории**
- Получение списка категорий с подкатегориями

**Продукты**
- Получение списка продуктов

**Корзина**
- Получить все продукты из корзины
- Добавление, изменение количества, удаление продукта в корзине


Для вывода результатов запросов включена пагинация (вывод результаов разбит постранично), результаты выводятся по 5 элементов на один запрос.

### Как запустить проект:

Клонировать репозиторий и перейти в него в командной строке:

```
git clone https://github.com/Andrey-Dubov-25/grocery_store
```

```
cd grocery_store/gcrocery_store
```

```
Cоздать и активировать виртуальное окружение:

```

```
python3 -m venv venv
```

```
source venv/Scripts/activate
```

Установить зависимости из файла requirements.txt:

```
pip install -r requirements.txt
```

Выполнить миграции:

```
python3 manage.py migrate
```

Запустить проект:

```
python3 manage.py runserver
```


**Через Docker**

Клонировать репозиторий и перейти в него в командной строке:

```
git clone https://github.com/Andrey-Dubov-25/grocery_store
```

```
cd grocery_store
```

Запустить контейнеры:

```
docker compose up --build
```

Собрать статику:

```
docker compose exec grocery_store python manage.py collectstatic --noinput
```

```
docker compose exec grocery_store cp -r /app/collected_static/. /grocery_store_static/static/ 

```
Применить миграции:

```
docker compose exec grocery_store python manage.py migrate
```

Создать суперпользователя:

```
docker compose exec grocery_store python manage.py createsuperuser
```

**Запуск тестов**

Перед запуском тестов убедитесь, что вы находитесь в корневой директории проекта grocery_store и DEBUG=True, после чего выполните команду:


```
cd grocery_store
```

Запуск тестов на pytest:

```
pytest
```

Запуск flake8:

```
flake8
```
